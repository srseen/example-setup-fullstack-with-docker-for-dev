# Set up Dockerfile and docker-compose for dev
### client
*  react+vite(ts)
### server
* express(ts)
### database
* mongodb

## git clone
```
git clone https://github.com/srseen/example-setup-fullstack-with-docker-for-dev.git
```
## lib install
* clinet
```
cd client
npm install
```
* server
```
cd server
npm install
```
